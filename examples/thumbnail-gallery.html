This example demonstrates how to open a gallery of images in Shadowbox by clicking on a thumbnail of any image in the gallery. As in the simple thumbnail example each thumbnail is wrapped in a link element with a rel="shadowbox" attribute. However, to ensure that Shadowbox displays all images in the same gallery, we can use the special square bracket notation to indicate the name of a gallery. Then, when any image in that gallery is clicked, all will be opened.

It is important to note that when using this method the gallery order will reflect the same order that the elements are found in the DOM.

This file is part of Shadowbox.js.
